# Hull_vs_Hull-less
Boundary removal and behavioral effects
This repo is a collection of small coding projects exploring what happens when you remove the "hull", the boundary conditions that normally stabilize the system. 
The idea is inspired by growing mushrooms and the way mycelium runs when you use hull-on versus hull-less millet as your seed spawn. 
Each project in this repo implements a system in two modes:
*Hull-on: the system operated with its normal constraints, rules, or validity checks in place.
*Hull-less: the constraints are removed or relaxed, allowing the system to enter states or behaviors that would normally be invalid or impossible. 
The goal is to observe, create some chaos, and hopefully observe some weird emergent behaviors. 
This is a working notebook, not a polished project.
